Supported Platforms

‚ùì Frequently Asked Questions

Is Fact-It really free?

Yes! Fact-It is 100% free with 100 fact-checks per day. You can optionally add your own AI API keys for unlimited checks.

Do I need to create an account?

No registration required! Just install and start using immediately.

What data does Fact-It collect?

None! We don't collect personal data, browsing history, or any identifying information. Everything is stored locally in your browser.

How accurate is the fact-checking?

Fact-It uses Groq AI (Llama 3.3 70B) with real-time web search. Accuracy depends on available sources. Always use critical thinking!

Can I use Fact-It for my business?

Yes! The free version works for everyone. We're planning a Premium Business tier with advanced features (coming soon).

How do I increase my daily limit?

You can add your own API keys from Groq, OpenAI, Anthropic, or Perplexity in the extension settings for unlimited checks.

What languages are supported?

Currently: Bulgarian and English. More languages coming soon!

Is my data private?

Absolutely! We only send post text to AI providers for fact-checking. No personal data, no browsing history, no tracking.

üîß Troubleshooting

Fact Check button not appearing

Solution: Refresh the page (Ctrl+R or Cmd+R). If still not working, try disabling and re-enabling the extension.

"Daily limit reached" message

Solution: Wait until midnight (resets daily) or add your own API key in Settings for unlimited checks.

Extension not working on specific website

Solution: Some websites block extensions. Try refreshing the page or check if the site is in our supported list.

Slow fact-checking results

Solution: AI processing takes 3-5 seconds. If longer, check your internet connection or try again later.
